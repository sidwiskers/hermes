# Support

Use the issue forms for reproducible Hermes bugs and focused feature requests.
Include the Hermes version, Go version, delivery mode, and a minimal sanitized
reproduction. General Telegram platform behavior should first be checked
against the [official Bot API documentation](https://core.telegram.org/bots/api).

Security vulnerabilities must be reported privately as described in
[`SECURITY.md`](SECURITY.md). Never publish bot tokens, webhook secrets, private
identifiers, or credential-bearing file URLs.

Hermes is maintained on a best-effort basis. A response-time commitment applies
only to security reports where `SECURITY.md` states one explicitly.
